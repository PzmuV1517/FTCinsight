# FTC Insight

Data analytics platform for FIRST Tech Challenge using the Expected Points Added (EPA) rating system.

## About

FTC Insight provides live-updating EPA statistics for FTC teams and events. The platform includes:

- **EPA Ratings** - Predictive team performance metrics
- **Match Predictions** - Win probabilities and score predictions
- **Event Analytics** - Strength of schedule, rankings, and insights
- **Historical Data** - Statistics from 2019 to present

## Tech Stack

- **Frontend**: Next.js, TypeScript, TailwindCSS
- **Backend**: FastAPI, Python, SQLite
- **Data Source**: FTC Events API
